# User Guide - Creator Connect

Welcome to Creator Connect! This guide will help you get started with managing your agency, creators, campaigns, and tasks.

## 🚀 Getting Started

### First Time Setup

1. **Create Your Account**
   - Sign up with email and password
   - Verify your email address
   - Complete your profile information

2. **Join or Create an Agency**
   - **New User**: You'll be prompted to create an agency
   - **Invited User**: Accept invitation from agency admin
   - **Agency Setup**: Add agency name and description

3. **Complete Your Profile**
   - Go to Settings → Profile
   - Add first name, last name
   - Upload profile picture (optional)
   - Configure notification preferences

## 🏢 Agency Management

### For Agency Admins

**Invite Team Members:**
1. Go to Settings → Agency Members
2. Click "Invite Member"
3. Enter email address and select role
4. Send invitation

**Manage Permissions:**
- **Admin**: Full access to all features
- **Member**: Limited access, cannot invite others

### For Team Members

**Joining an Agency:**
1. Check email for invitation
2. Click invitation link
3. Sign up or sign in
4. Automatically added to agency

## 📋 Task Management

### Creating Tasks

1. **Quick Task Creation:**
   - Use the "+" button in the task sidebar
   - Enter task title and press Enter
   - Task is created and assigned to you

2. **Detailed Task Creation:**
   - Click "+" button for task options
   - Add description and due date
   - Assign to team member (admins only)
   - Set priority level

### Managing Tasks

**Task Actions:**
- **✓ Complete**: Click checkbox to mark done
- **✏️ Edit**: Click pencil icon to modify
- **🗑️ Delete**: Click X to remove
- **↕️ Reorder**: Drag and drop to reorganize

**Task Assignment:**
- Admins can assign tasks to any team member
- Members can only create tasks for themselves
- Assignees receive notifications

### Task Workflow

```
Create Task → Assign (optional) → Work on Task → Mark Complete
     ↓              ↓                ↓             ↓
  Added to      Notification    Track Progress  Notification
   List           Sent                          Sent
```

## 👥 Creator Management

### Adding Creators

1. Go to **Creators** page
2. Click **"Add Creator"**
3. Fill in creator information:
   - Name and contact details
   - Social media handles
   - Location and notes
4. Save creator profile

### Creator Profiles

**Essential Information:**
- **Contact**: Email and phone
- **Social Media**: Instagram, TikTok, YouTube handles
- **Location**: Geographic location
- **Notes**: Internal notes and preferences
- **Status**: Active/Inactive status

### Managing Creator Relationships

- **Search**: Use global search to find creators
- **Filter**: Sort by activity status
- **Update**: Edit creator information as needed
- **Track**: Monitor creator activity and engagement

## 📊 Campaign Management

### Creating Campaigns

1. Navigate to **Campaigns** page
2. Click **"New Campaign"**
3. Set up campaign details:
   - Campaign name and description
   - Start and end dates
   - Budget allocation
   - Campaign status

### Campaign Lifecycle

**Campaign Statuses:**
- **Draft**: Planning phase, not yet active
- **Active**: Currently running campaign
- **Paused**: Temporarily stopped
- **Completed**: Finished campaign

### Campaign Tracking

- **Budget Monitoring**: Track spending vs budget
- **Timeline Management**: Monitor start/end dates
- **Performance Metrics**: View campaign analytics
- **Creator Assignment**: Link creators to campaigns

### Event Management

## 🔔 Notifications

### Notification Center

**Accessing Notifications:**
- Click bell icon in header
- Red badge shows unread count
- Dropdown shows recent notifications

**Notification Types:**
- **📋 Task Assignment**: When tasks are assigned to you
- **✅ Task Completion**: When assigned tasks are completed
- **📢 Campaign Alerts**: Campaign milestone updates
- **📅 Custom Events**: Event reminders and updates

### Managing Notifications

**Notification Actions:**
- **Mark as Read**: Click notification to read
- **Delete**: Click trash icon to remove
- **Mark All Read**: Bulk action for all notifications

### Notification Preferences

**Configuring Settings:**
1. Go to **Settings → Notifications**
2. Toggle notification types:
   - Email Notifications
   - Campaign Alerts
   - Weekly Reports  
   - Creator Updates
3. Save preferences

## 🆘 Troubleshooting

### Common Issues

**Login Problems:**
- Check email and password accuracy
- Verify account activation
- Reset password if needed
- Clear browser cache

**Notification Issues:**
- Check notification preferences
- Verify email address is correct
- Look for notifications in spam folder
- Refresh browser/app

**Performance Issues:**
- Check internet connection
- Clear browser cache
- Disable browser extensions
- Try different browser

**Data Not Loading:**
- Refresh the page
- Check agency membership status
- Verify permissions with admin
- Contact support if persistent

### Getting Help

**Self-Service:**
1. Check this user guide
2. Review notification preferences
3. Verify account settings
4. Try basic troubleshooting

**Contact Support:**
- Create support ticket
- Include error details and screenshots
- Describe steps to reproduce issue
- Provide account information
